What is CSS

  1. CSS stands for Cascading Style Sheets.

  2. CSS describes how HTML element are to be displayd on screen, paper, or in other media.

  3. CSS is not neccesaryCSS saves a lot of work. It can control the layout of multiple web pages all at once

  4. External stylesheets are stored in CSS files.

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements1
  • Internal - by using a style element in the head section
  • External - by using the line element to link to an external CSS file.
  • The most common way to add CSS. is to keep the style in external CSS file. However, this tutorial we will use inline add internal styles, because this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies a style to unique element. ID is also spacial in the you can use a special URL to link dorectly to an element and it's used by javaScript.

In CSS, selectors are used to traget a specific element or range of elements on a web page. Once an element hsa been targeted, a style or set of styles can be applied to the element.

There is a wide range of selectors available Two of the most commonly used are class and ID. Both are used to target element to which a style should be applied.
